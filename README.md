# Portfolio Mahasiswa Magang

Website portfolio modern, profesional, dan responsive untuk mahasiswa magang.

## Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript ES6
- Bootstrap 5.3
- Bootstrap Icons
- AOS (Animate On Scroll)
- Google Font: Poppins

## Struktur File

```
portfolio/
│
├── index.html
├── assets/
│   ├── css/
│   │   └── style.css
│   ├── js/
│   │   └── script.js
│   ├── img/
│   └── files/
│
└── README.md
```

## Fitur

- Navbar sticky dengan scroll effect
- Hero section dengan typing effect
- About section
- Skills section dengan progress bar
- Projects section
- Certificate section
- Contact section dengan form validation
- Footer
- Animasi scroll dengan AOS
- Dark mode toggle
- Back to top button
- Scroll progress bar
- Preloader
- Ripple effect pada button
- Responsive design

## Cara Menggunakan

1. Clone atau download repository ini
2. Buka file `index.html` di browser
3. Sesuaikan konten sesuai kebutuhan:
   - Ganti nama, informasi pribadi, dan foto
   - Tambahkan proyek dan sertifikat Anda
   - Ganti link social media dan kontak
   - Letakkan CV Anda di folder `assets/files/`

## Kustomisasi

### Warna

Anda dapat mengubah warna tema di file `assets/css/style.css` pada bagian CSS Variables:

```css
:root {
    --bg-color: #ffffff;
    --primary: #10b981;
    --secondary: #047857;
    --text-color: #1f2937;
    --gray: #6b7280;
    --border-color: #e5e7eb;
    --card-bg: #f9fafb;
    --hover: #059669;
}
```

### Font

Font yang digunakan adalah Poppins dari Google Fonts. Anda dapat mengubahnya di file `index.html`.
